Edusight is an accessibility-first learning app designed for
visually impaired, low-vision, and color-blind users.

## About
This project is a high-fidelity prototype built during an ideathon.
The focus is on user experience, accessibility, and system design.

## Features
- Voice-first interaction
- Eye-safe, high-contrast UI
- Minimal and distraction-free design
- Accessibility-focused user flow

## Tech Overview
- React (Vite)
- Component-based architecture
- Tailwind CSS
- AI-assisted rapid prototyping
